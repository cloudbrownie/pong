This is Pong.

Controls are:
 - UP ARROW: Moves the left paddle up
 - DOWN ARROW: Moves the left paddle down
 - ENTER / RETURN: Exits title and goes to game
 - ESCAPE: Quits to title if in game, closes the window if in menu.

To Play:
 - Press ENTER / RETURN to exit the title, the game will start immediately
 - Use up and down arrow keys to move your paddle, which is the left paddle, in order to bounce the ball 
 - Rounds reset after ball is scored
 - Exit the game by pressing ESCAPE
